Configuration

Global site parameters 🔗

On top of Hugo global configuration, Dot lets you define the following parameters in your config.toml (here, values are default).

Note that some of these parameters are explained in details in other sections of this documentation.

1[params]
2  # Prefix URL to edit current page. 
3  # Useful to give opportunity to people to create merge request for your doc.
4  # See the config.toml file from this documentation site to have an example.
5  # Javascript and CSS cache are automatically busted when new version of site is generated. 
6  # Order sections in menu by "weight"
7  # Call to action is default enabled, if you want to disable it. just change the 
8  enable = false
9  # You can change banner title and other text from the config file.
